Formularis en HTML

En HTML, els formularis són una forma de recopilar informació dels usuaris de la pàgina web. Permeten als visitants enviar dades a través del navegador web a un servidor web per processar-les. Els formularis contenen camps de text, caselles de selecció, botons d'opció, botons d'enviament, etc. que els usuaris poden omplir i enviar. Els formularis són una eina important per a la interacció amb els usuaris i s'utilitzen per a una àmplia gamma d'aplicacions, com ara registres d'usuari, enquestes, pagaments, etc. Així, els formularis són una part crucial de qualsevol pàgina web que requereixi interacció amb l'usuari.

Etiquetes de formulari

  • <form>: S'utilitza per crear un formulari HTML. Conté tot el contingut del formulari i defineix l'acció que s'ha de realitzar quan es lliura el formulari.
  • <input>: Serveix per crear camps d'entrada en el formulari. Els diferents tipus d'atributs type determinen el tipus d'entrada, com ara text, contrasenya, correu electrònic, etc. Aquests són alguns dels tipos de input:
    • text: Crea un camp d'entrada de text on l'usuari pot escriure text lliurement.
    • password: Crea un camp d'entrada de contrasenya on el text escrit es mostra com a caràcters ocults.
    • email: Crea un camp d'entrada per a l'adreça de correu electrònic. Pot comprovar que l'entrada sigui una adreça de correu vàlida.
    • number: Crea un camp d'entrada per a valors numèrics. Permet només l'entrada de números i pot tenir atributs addicionals com min, max i step per especificar els valors vàlids i el pas.
    • checkbox: Crea una casella de selecció on l'usuari pot marcar o desmarcar una opció. Pot tenir l'atribut checked per especificar que està inicialment marcat.
    • radio: Crea un botó d'opció on l'usuari pot seleccionar una única opció d'un grup d'opcions. Tots els botons d'opció en un grup han de tenir el mateix atribut name perquè només es pugui seleccionar una opció.
    • file: Crea un camp d'entrada per seleccionar un fitxer del dispositiu de l'usuari. Permet carregar fitxers des de l'ordinador.
    • date: Crea un camp d'entrada per a la selecció de dates. Pot mostrar un calendari o un selector de dates, depenent del navegador.
    • submit: Crea un botó d'enviament per enviar el formulari. Quan l'usuari fa clic en aquest botó, s'envia el formulari al servidor.
  • <textarea>: S'utilitza per crear un àrea de text multilínia en el formulari, on l'usuari pot escriure text extens.
  • <select>: Es fan servir per crear un desplegable d'opcions en el formulari. Les opcions s'afegeixen utilitzant l'etiqueta <option>
  • <button>: És per crear un botó en el formulari. Pot tenir diferents atributs
  • <label>: S'utilitza per associar una etiqueta amb un camp d'entrada. Proporciona una descripció o una etiqueta per al camp d'entrada relacionat.
  • <fieldset>: S'utilitza per agrupar elements del formulari junts en un conjunt lògic. Pot contenir etiquetes <legend>
  • <legend>: Per proporcionar un títol o una descripció per al grup de camps.
  • "action" (atribut): L'atribut action de l'etiqueta <form> especifica l'URL on s'enviarà la informació del formulari quan es lliuri.
  • "method" (atribut): L'atribut method de l'etiqueta <form> especifica el mètode HTTP a utilitzar per enviar la informació del formulari, com GET o POST.
