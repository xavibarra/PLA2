Llistes en HTML

En HTML tenim diverses manera de fr llistes en aquest document veurem exemples d'aquestes i l'estructura per fer-les.

Llistes desordenades

Les llistes desordenades en HTML són un tipus d'estructura de contingut que s'utilitzen per organitzar informació en forma de llista, sense seguir un ordre específic. Aquesta estructura es crea utilitzant l'etiqueta <ul> (unordered list) en HTML, que conté una sèrie d'elements <li> (list item) que conformen la llista. Cada element <li> es mostra com a un punt o una marca de bala davant del text del llistat. Les llistes desordenades són útils per a organitzar informació de manera clara i jerarquitzada, sense necessitat de seguir un ordre específic.

Exemple:

Llistes ordenades

Les llistes ordenades en HTML són un tipus d'estructura de contingut que s'utilitzen per organitzar informació en forma de llista, seguint un ordre específic. Aquesta estructura es crea utilitzant l'etiqueta <ol> (ordered list) en HTML, que conté una sèrie d'elements <li> (list item) que conformen la llista. Cada element <li> es mostra amb un número o una lletra davant del text del llistat, segons el tipus de llista ordenada que s'hagi especificat. Les llistes ordenades són útils per a organitzar informació de manera clara i jerarquitzada, seguint un ordre específic.

Exemple:

Classificació lliga Santander:

  1. Barça
  2. Madrid
  3. At. Madrid
  4. Real societat

Estructura:

<p>Classificació lliga Santander:</p>
<ul>
<li>Barça</li>
<li>Madrid</li>
<li>At. Madrid</li>
<li>Real societat</li>
</ul>

Atribut: type

En les llistes ordenades podem posar l'atribut type per escollir la manera amb la que ordenem les llistes. Aquests són els principals tipos:

  • 1: Llistes ordenades de forma decimal
  • Reverse: Llista ordenada al revés
  • A: Ordenada alfàbticament amb majúscules
  • a: Ordenada alfàbeticament amb minúscules
  • I: Ordenada amb números romans en majúscula
  • i: Ordenada amb números romans en minúscula

Llistes amb descripció

Les llistes amb descripció en HTML són un tipus d'estructura de contingut que s'utilitzen per organitzar informació en forma de llista, amb un títol o una descripció per a cada element de la llista. Aquesta estructura es crea utilitzant l'etiqueta <dl> (description list) en HTML, que conté una sèrie d'elements <dt> (description term) per als títols o descripcions, i elements <dd> (description data) per a la informació relacionada amb cada títol o descripció. Les llistes amb descripció són útils per a presentar informació de manera clara i estructurada, especialment en casos on es necessita una descripció detallada de cada element de la llista.

Exemple:

Llenguatjes

HTML
HyperText Markup Language
CSS
Cascading Style Sheets
JS
JavaScript

Estructura:

<p>Llenguatjes</p>
<dl>
<dt>HTML</dt>
<dd>HyperText Markup Language<dd>
<dt>CSS</dt>
<dd>Cascading Style Sheets<dd>
<dt>JS</dt>
<dd>JavaScript<dd>
</dl>
