HTML

Les etiquetes en HTML són elements utilitzats per definir l'estructura i el contingut d'una pàgina web. Cada etiqueta s'encapsula entre < i > i pot tenir atributs opcionals. Les etiquetes serveixen per indicar al navegador com interpretar i mostrar el contingut.

Les etiquetes permeten crear diferents tipus de components com títols (<h1> a <h6>), paràgrafs (<p>), enllaços (<a>), imatges (<img>), llistes (<ul>, <ol>, <li>), taules (<table>, <tr>, <td>), formularis (<form>, <input>, <select>, <textarea>), entre d'altres.

A més, les etiquetes poden contenir altres etiquetes, permetent l'anidament i l'organització jeràrquica del contingut. Amb l'ús adequat de les etiquetes, es pot definir l'estructura, el format i l'accessibilitat de la pàgina web, proporcionant una experiència coherent i comprensible per als usuaris.
